## 识别车标

### 简介
在学习 TensorFlow 的过程中，我们在开源社区中找到了一个名为 MobileNet 的模型，该模型能对图像进行分类。Google 发布的 tensorflow-for-poets 项目正好支持这个模型，我们希望能结合 TensorFlow 与 MobileNet，将其应用于分类不同车标的图片。同时以此作为基础，体验 ML/TF 中不同超参数对训练的影响。
